# トラブルシューティングガイド

このドキュメントはトラブル発生時に参照する。通常のセッションでは読み込み不要。

## 基本方針

1. 可能性が高い原因から順に確認する
2. ユーザーへの確認依頼は最小回数にまとめる
3. 確認の指示にあたっては、何を開いて、どこの何をクリックするか、具体的にナビゲートすること
4. ユーザーが状況を説明できない場合は、スクリーンショットの送付を依頼する

## スクリーンショットの撮り方

ユーザーが「わからない」と回答した場合や、状況の説明が難しい場合は、以下の手順を案内すること:

> **画面のスクリーンショットを撮って送ってください:**
> 1. キーボードの **Win + Shift + S**（Windows）または **Cmd + Shift + 4**（Mac）を押してください
> 2. 確認したい部分をマウスでドラッグして範囲選択してください
> 3. このチャット欄をクリックし、**Ctrl + V**（Windows）または **Cmd + V**（Mac）で貼り付けてください
